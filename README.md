# back

## Instalar Dependencias
```
pip install -r requirements.txt
```

### Iniciar migrações
```
python manage.py migrate
```
### Gerar arquivos estáticos
```
python manage.py collectstatic
```

### Iniciar aplicação em modo desenvolvimento
```
python manage.py runserver
```

### Acessar a aplicação

```
http://localhost:8000/
```
